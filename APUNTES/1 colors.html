Uso de colores en CSS

Múltiples usos:

RGB + Opacidad: Uso actual y más recomendado

Estructura -> color:rgb(12 12 12 / 0.2)

R: Red, G: Green, B: Blue

Ejemplo

HSL: Segunda alternativa

Estructura -> color:hsl(50 50 50)

H: Referencia al matiz, S: Saturation, L: Luminosity

Ejemplo

OKLCH: El posible futuro del color

Estructura -> color: oklch(20 20 173)
o tambien puede ser -> color: oklcha(20 20 173 / 20%) incluyendo opacidad

L: Lightness, C: Chroma, H: Hue angle, A: Opacity

Ejemplo
